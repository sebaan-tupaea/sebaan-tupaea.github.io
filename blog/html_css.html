What is the difference between Margin, Border, and Padding?

Conceptualising CSS - Who is she?

Margin, border, and padding are style features that help manage the space and arrangement of elements on a webpage.

  1. Margin

    2. What it does: Adds space outside the edge of an element, pushing it away from other things.

    Example: If you have a picture with a margin, it will move away from nearby text or other images.

  2. Border

    3. What it does: Adds a line around the edge of an element. Itâ€™s like drawing a frame around a picture.

    Example: If you put a border around a button, it will have a visible line around it that can be styled in different colours or thicknesses.

  3. Padding

    4. What it does: Adds space inside the edge of an element, between the content and the border.

    Example: If you have a button with padding, the text inside it will be pushed away from the edge, making it look nicer and easier to read.
